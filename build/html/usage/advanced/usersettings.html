User SettingsÂ¶

All user settings, selections and default board dimensions are customizable to model are saved external to the model. The only change to the model will occur when MBP creates a map with bodies as a map object in the browser tree. This results with a single entry in the UnDo list.

The created map can be removed by performing an UnDo, CMD Z on MacOS or Ctrl Z on Windows. Any board dimension, settings or selections will not be undone when the UnDo is performed.
